# 💊 MediCare Reminder

A modern medical reminder web application that helps users manage medicines, set reminders, track medication history, and monitor daily progress.

---

## 🚀 Features

### 👤 User Management
- User registration
- Login system
- Profile management
- Logout functionality

### 💊 Medicine Management
- Add medicines
- Edit medicines
- Delete medicines
- Search medicines
- View medicine schedules

### 🔔 Smart Reminders
- Browser notification reminders
- Medicine time alerts
- Reminder preferences

### 📊 Health Tracking
- Medicine completion tracking
- Taken/Missed history
- Progress chart

### ⚙ Settings
- Notification settings
- Dark mode option

---

## 🛠 Technologies Used

### Frontend

- HTML5
- CSS3
- JavaScript
- Chart.js
- Local Storage
